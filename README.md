# FileFlippy

**FileFlippy** is a simple image converter that works right in your browser, no server so files are kept safe on your device

## How to use it

1.  Go to [fileflippy.netlify.app](https://fileflippy.netlify.app/)
2.  Choose or drop the image you want to convert
3.  Select the new format (like PNG, JPG, or WEBP)
4.  Download your new image!
